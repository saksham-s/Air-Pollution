The tweets have been marked on the world map using Google's api and hovering over the marker gives the user's screen name. The geo-location field has been used to get the location of the tweet and as very few tweets have this enabled, the number of markers is comparatively small.

Sentiment Analysis

The sentiment of the tweets has been analysed using a python library (afinn). The more positive a value the more positive the sentiment is and vice versa. A value of 0 means a neutral tweet. The counts for the tweets having positive, negative and neutral sentiment have been calculated and displayed using pie-chart.
